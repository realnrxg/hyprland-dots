## SET UP
- **OS:** Arch Linux
- **WM:** Hyprland
- **APP LAUNCHER:** Rofi
- **TERMINAL:** Kitty
- **SHELL:** Fish
- **NOTIFICATIONS:** Dunst
- **BAR:** Waybar
- **WALLPAPER:** Waypaper & mpvpaper
- **LOCK:** gtklock

## DEPENDENCIES
- hyprland,hyprshot,waybar,dunst,fish,rofi,waypaper,mpvpaper,ssdm,gtklock,pavucontrol,ttf-jetbrains-mono-nerd,ghostty

# IF U WANNA USE SOMETHING ELSE THAN GTKLOCK

go to '~/.config/waybar/config'
find "custom/power"
under "lock":
change from "gtklock" to ur desired lock screen manager
